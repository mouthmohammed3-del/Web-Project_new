Web-Project_E-learning

Description

This project is created to learn and practice JavaScript by building a simple E-learning web application.

The project focuses on understanding JavaScript fundamentals and applying them in a real web project, including:

- Variables and Data Types
- Functions
- DOM Manipulation
- Events
- Arrays and Objects
- ES6 Features
- Form Validation
- Local Storage

Technologies Used

- HTML5
- CSS3
- JavaScript

Purpose

The main goal of this project is to improve JavaScript skills through hands-on practice while developing an interactive E-learning website.
